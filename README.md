## 🎯 What is ScanTire?

ScanTire uses computer vision and deep learning to automatically inspect tires and provide detailed reports — no expensive equipment needed, just a smartphone camera.

### Key Features:

- **📏 Tread Depth Measurement** — Measures remaining tread using TWI (Tread Wear Indicators) as reference points
- **🔤 Tire Size Recognition (OCR)** — Reads embossed markings from curved sidewall surfaces
- **❄️ Type Classification** — Identifies Summer/Winter/All-Season/Off-Road tire types
- **📊 Wear Grading** — AI grades condition into 5 classes (New → Critical)
